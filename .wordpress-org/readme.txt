Clx.bz Integrator easily integrates the automatic link shortening functionality to your website.

== Description ==

Clx.bz offers website owners a simple way to shorten and track clicks on your links.

The Clx.bz Integrator Plugin - simplfies the integration process. Using the plugin you will easily be able to set the 
link clases you would like to shorten and select which page you would like to run the automatic link shortner on.

== Installation ==

Installing the plugin is easy. Simply follow these steps:

1. Upload `clx-bz-integrator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the settings page to setup the plugin
